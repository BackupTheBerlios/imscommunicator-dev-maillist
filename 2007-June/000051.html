Yes they are different because the IMS-Communicator is using an old version of the jain-sip stack (JAIN-SIP API 1.1 implemented by NIST-SIP 1.2) with some changes made by us, which later were adapted to the new JAIN-SIP API and RI 1.2, currently available at the dev.java.net project.
I would recommend you to use the more recent one since the one that the IMS-Communicator is using is no longer officially supported... Anyway, I could provide the source code if you wish...
